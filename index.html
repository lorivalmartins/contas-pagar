☁️ Configurar Nuvem — JSONBin.io
Cole aqui as credenciais do JSONBin para que todos que abrirem o sistema vejam os mesmos dados em tempo real, sem precisar trocar arquivos.
📋 Como configurar (passo a passo)
1 Acesse jsonbin.io e clique em Sign Up para criar uma conta gratuita
2 Após fazer login, clique no seu perfil (canto superior direito) → API Keys → copie a Master Key
3 No menu lateral clique em Bins → Create Bin → cole {} como conteúdo → clique Create
4 Copie o Bin ID que aparece na URL ou no topo da tela (ex: 64a1b2c3...)
5 Cole a Master Key e o Bin ID nos campos acima → clique Conectar e Sincronizar
6 Compartilhe o arquivo HTML com a outra pessoa. Ela cola as mesmas credenciais e pronto — dados sincronizados!
⚠️ Importante: A Master Key dá acesso total aos seus dados. Compartilhe apenas com pessoas de confiança. O plano gratuito do JSONBin suporta até 10.000 requisições/mês — mais que suficiente para uso diário da equipe.
