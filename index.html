☁️ Configurar Nuvem — JSONBin.io
Com a nuvem configurada, todos que abrirem o sistema veem os mesmos dados em tempo real — sem trocar arquivos.
Passo a passo
1Acesse jsonbin.io → Sign Up (gratuito)
2Login → clique no perfil → API Keys → copie a Master Key
3Menu lateral → Bins → Create Bin → cole {"contas":[],"contaIdCounter":1} → Create
4Copie o Bin ID da URL (ex: 64a1b2c3...)
5Cole os dados acima e clique Conectar
6Compartilhe o HTML + Master Key + Bin ID com sua equipe. Todos verão os mesmos dados!
